# Discord Color Picker

This helps you preview your name color before changing it. I made it because of gradient colors and wanted a way to check them before applying.

## Features

- Name preview
- Three color modes: palette, custom, gradient
- Responsive and mobile-friendly interface
- Random color and gradient generator
- Hex color input support

## Run Locally

Clone the project

```bash
  git clone https://github.com/gentrit04/discord-color-picker.git
```

Go to the project directory

```bash
  cd discord-color-picker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
